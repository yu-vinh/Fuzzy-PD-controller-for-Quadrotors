# Fuzzy-PD-controller-for-Quadrotors (Matlab r2025a)

In this case, i simulate the fuzzy with 2-in 1-out structure and use the 2 parallel branch for FIS of delta Kp and delta Kd.
About the environment of simulation, with dx=dy=dz=d_phi=d_theta=d_psi = 0 (errors), the results is very good with the discrepancies of desired trajectory and the calculated trajectory is very small( approximate 0.01%)
